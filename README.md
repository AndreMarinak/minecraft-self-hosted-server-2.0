# Minecraft Self Hosted
This repository contains scripts and configuration for self hosting a Minecraft server on a linux machine. You just need some type of machine/pc to be able to follow this tutorial. It will use linux for the OS.

## Knowing the basics
- Knowing basic terminal commands like ls, mkdir, cd, nano, cat will make this much easier


## Getting Started
Choose the guide that matches your situation:

- I already have a Linux machine or server setup
  → README-startfromlinux.md

- I do NOT have a Linux machine yet, I just have an old pc/machine
  → README-startfromzero.md
